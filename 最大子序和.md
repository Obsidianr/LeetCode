求最大子序和，
初始化sum等于数组中的第一个数，遍历数组，

如果子序和大于0，就将下一个数加入到sum中，设置max记录最大的子序和

如果sum小于就等于和当前数比较取较大者，



```java
 
 public int maxSubArray(int[] nums) {
        int sum = nums[0];
        int max = nums[0];
        for(int i = 1; i < nums.length;i++){
            if(sum>= 0){
                sum += nums[i];
            }
            else
                {
                    sum = Math.max(sum,nums[i]);
                }
            max = Math.max(sum,max);
        }
        return max;
    }
```
